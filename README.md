# DollarPlan - a budgeting tool #

## Before running for the first time
`npm install`

## starting DollarPlan ##

### Development
1. `npm run watch`.
2. In a separate terminal: `npm start`.

App will automatically reload when changed.

### Production
1. `npm run build`
2. `npm start prod`

## testing DollarPlan ##
`npm test`
